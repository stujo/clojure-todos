# Clojure Vectors

* [http://clojuredocs.org/clojure.core/vector](http://clojuredocs.org/clojure.core/vector)

```clojure
[1 2 3]
=> [1 2 3]
```

```clojure
(vector 1 2 3)
=> [1 2 3]
```


```clojure
(type (vector 1 2 3))
=> clojure.lang.PersistentVector
```

```clojure
(class (vector 1 2 3))
=> clojure.lang.PersistentVector
```
